# Project Guidelines

## Project Overview

This is a premium beauty atelier website built with:

* Next.js App Router
* TypeScript
* CSS Modules
* Vercel deployment

The project focuses on:

* luxury minimalist design
* excellent SEO
* high performance
* mobile-first UX
* PWA standards

---

# Design Principles

* Elegant and minimal UI
* Large spacing and typography-focused layouts
* Luxury beauty branding
* Black, gold, soft pink color palette
* Avoid cluttered interfaces
* Avoid excessive animations

---

# Performance Rules

* Prioritize Lighthouse scores above 95
* Keep bundle size minimal
* Avoid heavy dependencies
* Use native CSS whenever possible
* Optimize all images
* Prefer static rendering and server components

---

# SEO Rules

* SEO-first architecture
* Semantic HTML required
* Proper heading hierarchy
* Structured data (schema.org)
* Optimized metadata for all pages
* Local SEO for Kielce beauty services

Target keywords include:

* makijaż Kielce
* makijaż ślubny Kielce
* stylizacja włosów Kielce
* makeup artist Kielce

---

# Styling Rules

* Use CSS Modules only
* No Tailwind CSS
* No inline styles unless necessary
* Shared variables should be stored globally

---

# Component Architecture

Structure:

* components/ → reusable UI components
* sections/ → large homepage sections
* app/ → routing and pages

Keep components:

* reusable
* accessible
* lightweight
* mobile-first

---

# Accessibility

* Proper aria labels
* Keyboard accessibility
* Sufficient color contrast
* Semantic HTML elements

---

# Animation Rules

* Use subtle animations only
* Prefer CSS transitions
* Avoid animation-heavy libraries

---

# Code Quality

* Strong TypeScript typing
* Avoid unnecessary client components
* Prefer server components
* Keep files clean and modular
